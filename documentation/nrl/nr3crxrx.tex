\section{The R\textsc{exx}R\textsc{exx} class}\label{refrexxrexx}

The \code{RexxRexx} class contains a copy of every method of the \code{Rexx} class, but callable in a non-object-oriented manner, like is done in Classic R\textsc{exx}. An automatic \code{uses RexxRexx} is added to the generated Class due to the default \code{implicituses} commandline option.

\label{c2classic}In the cases of \code{c2d()} and \code{c2x()}, where the \nr{}
oo-notation only accepts a
\code{char} argument, these functions, when called in Classic notation,
accept a \code{Rexx} string argument, and will convert all characters
in the string argument to decimal or hexadecimal representations, respectively.
